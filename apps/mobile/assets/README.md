# Assets Directory

Place your app assets here:

- **`icon.png`**: App icon (1024x1024 recommended)
- **`splash.png`**: Splash screen (1284x2778 for iOS, adaptable for Android)
- **`adaptive-icon.png`**: Android adaptive icon (1024x1024)
- **`favicon.png`**: Web favicon (48x48 minimum)

## Quick Placeholders

For development, you can use placeholder images from:
- https://placehold.co/1024x1024/0B0E14/FFFFFF?text=Peaks
- https://via.placeholder.com/1024x1024/0B0E14/FFFFFF?text=Peaks

Or comment out the asset paths in `app.config.ts` temporarily.
